# 📘 Entregables - Aprendizaje Automático I

Este repositorio contiene los entregables correspondientes a la asignatura **Aprendizaje Automático I**, perteneciente a la **Especialización en Analítica y Ciencia de Datos**.

## 🎯 Objetivo

Desarrollar e implementar modelos predictivos aplicando técnicas de aprendizaje automático, con el fin de analizar datos y generar valor a partir de la información.
